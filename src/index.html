Google Chrome Bookmarks Converter

This tool allows converting Chrome's bookmarks JSON file to a more readable and usable format, e.g. for synchronization, comparison, finding duplicates, etc.

Usage: paste the contents of c:\Users\<name>\AppData\Local\Google\Chrome\User Data\Default\Bookmarks file to the field below and click Convert.
